# Simple React Redux CRUD Application

![App Screenshot](https://i.ibb.co/gWL6Gzt/dddddddddd.png)

## Features

- Add books
- Edit book details
- Delete books

## Technology Stack

- **REACT**
- **REDUX**
- **TAILWIND**

## How to Use

1. **Installation:**
   - Clone the repository
   - Run `npm install` to install dependencies

2. **Run:**
   - Execute `npm run dev` to run the application
